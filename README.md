## 🌟 Project Overview
**Recipe Ideas App** helps users discover new recipes based on ingredients they already have at home.  
It features:
- A clean and minimal black-and-white theme with elegant design.
- Ingredient-based recipe search (powered by TheMealDB API).
- Option for users to add their own recipes.
- Secure delete option (developer-only access using password).
- Inspirational “Food Thought of the Day.”
- Fully responsive layout for desktop and mobile.

## 🧠 Features
✅ Search recipes by ingredient (even one ingredient shows results).  
✅ Add your own custom recipe with title, image, and ingredients.  
✅ Developer-only delete option (password protected).  
✅ Clean navigation bar and footer for a professional look.  
✅ Smooth animations and modern responsive design.  

---

## 🧩 Tech Stack
- **Frontend Framework:** React (TypeScript)
- **Styling:** Tailwind CSS
- **API Used:** [TheMealDB API](https://www.themealdb.com/api/json/v1/1/filter.php?i={ingredient})
- **Deployment:** CodeSandbox
